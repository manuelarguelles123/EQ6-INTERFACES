Introducción al servidor EFT

Explora los temas clave para aprender sobre el servidor EFT y sus funcionalidades avanzadas.

Este curso está diseñado para proporcionarte las herramientas necesarias para aprender sobre el servidor EFT. A continuación, encontrarás los módulos, recursos y actividades que te ayudarán a alcanzar tus objetivos.

Módulos del Curso

Tema 1: Introduction to EFT Server

El servidor de transferencia de archivos mejorada (EFT) de GlobalSCAPE es un archivo administrado transfer...

  • ¿Qué es el servidor EFT? Una solución modular que permite a las empresas gestionar transferencias de datos de forma segura y rentable.
  • Principales características y beneficios: Alternativa a líneas arrendadas, VPNs y soluciones personalizadas, con opciones de escalabilidad para empresas de todos los tamaños.
  • Casos de uso comunes: Transferencias seguras entre socios comerciales, clientes y oficinas remotas.

Tema 2: Transferencias seguras de archivos

Datos está protegido y controlado durante el tránsito y el almacenamiento, y es totalmente auditable durante ese proceso...

  • Protocolos seguros: Protección de la propiedad intelectual, secretos comerciales, y los archivos de los clientes transferidos a través de Internet mediante protocolos que incluyen FTPS (SSL/TLS), SFTP (SSH2), HTTPS y AS2.
  • Implementación simple: Cumplimiento de los requisitos reglamentarios y de privacidad como PCI, DSS y HIPAA.
  • Mantenimiento automático de archivos programables: Automatiza acciones como supervisar carpetas, enviar notificaciones por correo electrónico y eliminar datos antiguos.

Tema 3: Seguridad ampliada del servidor EFT

Explora las opciones avanzadas de seguridad del servidor EFT, incluyendo módulos adicionales que mejoran la protección de los datos y la auditoría informática.

  • AS2 Módulo: Permite el intercambio seguro de datos empresariales mediante HTTP/HTTPS.
  • HS-PCI Módulo - Servidor: Facilita el cumplimiento de PCI DSS para proteger datos financieros y confidenciales.
  • Auditoría y Módulo de Informes (ARM): Genera informes detallados y realiza auditorías de actividades dentro de las organizaciones.

Tema 4: Apoyo

Aprende cómo acceder al soporte técnico del servidor EFT, incluyendo documentación, foros y asistencia directa para resolver problemas.

  • Documentación oficial: Guías de usuario y manuales técnicos para configurar y administrar el servidor EFT.
  • Foros de la comunidad: Espacios para resolver dudas y compartir experiencias con otros usuarios.
  • Asistencia técnica personalizada: Soporte por correo electrónico o teléfono para resolver problemas específicos.

Tema 5: ¿Qué hay de nuevo en el servidor EFT?

Descubre las últimas actualizaciones y características del servidor EFT, incluyendo mejoras en la seguridad, rendimiento y nuevas funcionalidades.

  • Mejoras en la seguridad: Nuevas herramientas para cumplir con estándares como PCI DSS y HIPAA.
  • Optimización del rendimiento: Actualizaciones para mejorar la velocidad y eficiencia de las transferencias.
  • Nuevas funcionalidades: Herramientas avanzadas para la gestión de usuarios y automatización de tareas.
